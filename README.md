# Real-Time Device Tracker 🚀

A real-time location tracking application built with Node.js, Express, and Socket.io. This project dynamically tracks connected devices and plots their live locations on an interactive map using Leaflet.js.

## ✨ Features
* **Live Location Tracking:** Captures device coordinates in real-time using the HTML5 Geolocation API.
* **Instant Map Updates:** Broadcasts location changes instantly to all connected clients using Socket.io.
* **Interactive Mapping:** Renders high-performance, mobile-friendly maps using Leaflet.js and OpenStreetMap tiles.
* **Dynamic Markers:** Automatically creates, updates, and removes map markers as users connect, move, and disconnect.

## 🛠️ Technologies Used
* **Backend:** Node.js, Express.js
* **Real-time Engine:** Socket.io
* **Frontend:** EJS (Embedded JavaScript templates), Vanilla HTML/CSS/JS
* **Map API:** Leaflet.js, OpenStreetMap
